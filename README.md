## ブランチ戦略

- main: メイン。実際に公開・運用する最前線
- hotfix: main に反映してから見つかった緊急対応が必要なもの
- release: 最終確認。校正やレイアウト微調整といったクオリティの最終調整
- develop: 機能を完全に形にする。feature から上がってきたものをフィードバック
- feature: 新機能の追加。草案の詳細を決めて実現って感じ
